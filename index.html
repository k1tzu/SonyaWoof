How to Buy

Ethereum

  1. Install a wallet (e.g., MetaMask) and switch to Ethereum Mainnet.
  2. Get ETH for gas and swapping.
  3. Open Uniswap with the SNYA address prefilled.
  4. Set slippage only if needed; confirm the swap.

Solana

  1. Install Phantom (or compatible) and ensure you are on Solana Mainnet.
  2. Get SOL for gas and swapping.
  3. Open Jupiter or Raydium (SOL pair) or Raydium (USDC pair).
  4. Review route, slippage, and confirm the swap.

Always verify the addresses before swapping.
